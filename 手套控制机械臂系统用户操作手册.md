## 系统概述

手套控制机械臂系统是一种基于手势识别的远程控制设备，通过佩戴智能手套，用户可以直观地控制机械臂完成各种动作。该系统采用 ESP32 作为主控芯片，集成了多种传感器，实现了精确的手势捕捉和机械臂控制。

### 主要功能特点

* **直观控制**：通过手势直接控制机械臂动作
* **实时响应**：10Hz 采样频率，快速响应手部动作
* **无线连接**：WiFi UDP 通信，支持远程控制
* **多自由度**：支持机械臂上下、左右移动和夹取动作

## 硬件组成

### 手套端硬件


| 部位   | 传感器类型  | 功能描述           |
| ------ | ----------- | ------------------ |
| 大拇指 | Flex 传感器 | 控制机械臂上下移动 |
| 食指   | Flex 传感器 | 控制机械臂夹取动作 |
| 中指   | MPU6050     | 控制机械臂左右移动 |
| 面包板 | MPU6050     | 控制机械臂夹轴转动 |
| 主控板 | ESP32       | 数据采集与指令发送 |

### 机械臂端硬件

* 机械臂本体（需支持相应控制接口）
* 控制板（需与手套端通信协议匹配）
* 电源适配器

## 系统连接

### 传感器连接示意图

```
ESP32

├── Flex1 (GPIO1) ── 大拇指传感器

├── Flex2 (GPIO2) ── 食指传感器

├── I2C接口 ── MPU6050 (地址0x68)

├── I2C接口 ── MPU6050 (地址0x69)

└── WiFi模块 ── 无线通信
```

### 连接步骤

1. **传感器安装**：

* 将 Flex 传感器分别固定在大拇指和食指位置
* 将 MPU6050 模块固定在中指和面包板指定位置
* 确保传感器与皮肤良好接触

1. **电路连接**：

* Flex 传感器输出端连接 ESP32 的 GPIO1 和 GPIO2
* MPU6050 通过 I2C 接口连接 ESP32
* 确保所有连接牢固可靠

1. **电源连接**：

* 为 ESP32 提供 5V 电源
* 确保电源稳定，避免电压波动

## 软件配置

### 开发环境准备

1. **安装 Arduino IDE**

* 下载并安装最新版本的 Arduino IDE
* 安装 ESP32 开发板支持包

1. **安装必要库**

* WiFi 库
* Wire 库（I2C 通信）
* 传感器驱动库

### 网络配置

修改代码中的 WiFi 配置参数：

```
// WiFi配置参数

static const char\* WIFI\_SSID  = "你的WiFi名称";

static const char\* WIFI\_PASS  = "你的WiFi密码";

static const char\* DEST\_IP    = "机械臂IP地址";

static const uint16\_t DEST\_PORT = 5005;
```

### 传感器校准

系统支持传感器零偏标定，确保在使用前进行校准：

1. 将手套平放，确保所有传感器处于自然状态
2. 上电后系统会自动进行 200 次采样的零偏标定
3. 标定过程中请勿移动手套

## 操作指南

### 基本手势控制


| 手势动作     | 机械臂响应     | 操作说明             |
| ------------ | -------------- | -------------------- |
| 大拇指弯曲   | 机械臂上升     | 弯曲程度控制上升速度 |
| 大拇指伸直   | 机械臂下降     | 伸直程度控制下降速度 |
| 食指弯曲     | 机械臂夹取     | 弯曲程度控制夹取力度 |
| 食指伸直     | 机械臂松开     | 完全伸直时完全松开   |
| 手腕左右转动 | 机械臂左右移动 | 转动角度控制移动距离 |
| 手臂上下摆动 | 夹轴转动       | 摆动幅度控制转动角度 |

### 操作步骤

1. **系统启动**：

* 给手套端和机械臂端上电
* 等待 WiFi 连接成功（约 3 秒）
* 观察指示灯状态确认系统正常

1. **姿态调整**：

* 保持手套在自然状态下初始化
* 系统会自动进行传感器校准

1. **控制操作**：

* 根据需要做出相应手势
* 机械臂会实时响应手部动作
* 操作完成后保持手势稳定

### 注意事项

* 操作时保持手部动作平稳，避免剧烈晃动
* 确保在有效通信范围内操作（通常 10-50 米）
* 定期检查传感器连接是否牢固
* 避免在强电磁干扰环境下使用

## 故障排除

### 常见问题及解决方案


| 问题现象       | 可能原因       | 解决方案                     |
| -------------- | -------------- | ---------------------------- |
| WiFi 连接失败  | 网络配置错误   | 检查 WiFi 名称和密码是否正确 |
| 传感器无响应   | 连接松动或损坏 | 检查传感器连接，更换损坏部件 |
| 机械臂动作异常 | 校准数据错误   | 重新上电进行零偏标定         |
| 控制延迟大     | 网络信号弱     | 靠近路由器或增加 WiFi 中继   |
| 系统不稳定     | 电源电压不稳   | 检查电源适配器输出电压       |

### 诊断方法

1. **串口调试**：

* 连接串口监视器（波特率 115200）
* 观察传感器数据输出是否正常
* 检查 WiFi 连接状态信息

1. **传感器测试**：

* 分别测试每个传感器的输出值
* 确认数据在合理范围内变化
* 检查滤波效果是否良好

## 技术参数

### 系统参数


| 参数项目 | 技术指标 |
| -------- | -------- |
| 主控芯片 | ESP32    |
| 采样频率 | 10Hz     |
| ADC 精度 | 12 位    |
| 通信方式 | WiFi UDP |
| 通信端口 | 5005     |
| 工作电压 | 5V       |
| 工作电流 | <200mA   |
| 工作温度 | 0-40°C  |

### 传感器参数

**Flex 传感器**：

* 电阻变化范围：10kΩ-50kΩ
* 弯曲角度范围：0-90°
* 响应时间：<10ms

**MPU6050**：

* 加速度计量程：±2g
* 陀螺仪量程：±250°/s
* I2C 通信接口
* 16 位 ADC 分辨率

### 软件参数

* 滤波方式：EMA 指数移动平均
* 采样次数：8 次平均
* 数据平滑：EMA 系数 0.2-0.3
* 异常检测：加速度阈值 0.2g，陀螺仪阈值 30°/s

## 维护保养

### 日常维护

1. **清洁保养**：

* 定期清洁传感器表面，保持良好接触
* 使用软布蘸清水擦拭，避免使用化学溶剂
* 清洁时断开电源，确保安全

1. **连接检查**：

* 每周检查一次电线连接是否牢固
* 检查传感器固定是否松动
* 及时更换老化或损坏的线缆

### 定期校准

建议每月进行一次系统校准：

1. 将手套放置在水平桌面上
2. 确保所有传感器处于自然状态
3. 重新上电进行零偏标定
4. 记录校准数据，建立维护档案

### 存储要求

* 存放在干燥通风的环境中
* 避免阳光直射和高温环境
* 长期不使用时建议每月通电检查一次
